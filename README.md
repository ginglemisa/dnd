# D&D 5e 2024 快速創角工具（主持人版）

這個專案是一個**前端單檔為主**的 D&D 5e（2024 Basic Rules）快速角色建立工具，定位是：
- 給玩家快速填角卡、輸出可讀資料。
- 給主持人（DM）快速核對角色關鍵數值與規則欄位。

> 對熟悉 5e 規則書邏輯的 DM 而言，你可以把它當成「角色資料整理器」，不是規則裁定引擎。

---

## 專案定位（給主持人）

- 角色資料集中在同一頁面，方便 Session 0 / 開團前審核。
- 支援常見 5e 欄位：能力值、熟練、豁免、技能、法術、裝備、生命值等。
- 法術清單以 Basic Rules 可用內容為主；不是 PHB 全收錄版本。

---

## 你可以期待它做好的事

- 快速建立 1~20 級角色的**結構化卡面資訊**。
- 自動更新部分衍生數值（例如能力調整值、部分職業相關欄位）。
- 提供法術欄位與筆記欄，讓玩家在開團前先整理常用資源。

---

## 你不該期待它代替的事（重要）

- **不代替 DM 裁定**：臨時優勢/劣勢、覆蓋規則、家規優先順序仍由主持人判定。
- **不等於規則資料庫**：內容可能只涵蓋 Basic Rules，非完整 PHB/擴充。
- **不保證所有 Build 邊角案例全自動正確**：多職業、特定專長互動等，仍建議人工複核。

---

## 法術顯示策略（目前）

- 介面預設顯示到**三環法術**（含戲法、一~三環）。
- 四環到九環區塊在主頁面不提供可視選單，避免誤以為可直接用隱藏指令解鎖。

---

## 建議主持人工作流

1. 玩家先在工具填完角色核心欄位。
2. 你先看：能力值、熟練加值、豁免熟練、AC、HP、主武器攻擊與傷害計算。
3. 再看法術欄位是否符合職業等級可接觸範圍。
4. 最後才處理背景敘事、裝備細節、開場資源（法術位/特性次數）。

這樣比逐行核對規則書更快，也比較不會漏掉「數值正確但規則來源錯誤」的問題。

---

## 開發與維護備註

- 本專案目前以 `index.html`（以及離線版本）承載主要邏輯。
- 若你要擴充到完整 PHB 法術或模組化結構，建議下一步：
  - 將 UI 與資料處理拆分；
  - 把規則來源（Basic/PHB/家規）做成可切換層；
  - 加入最小化自動測試（至少覆蓋能力值/法術顯示條件）。
