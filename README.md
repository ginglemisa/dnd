# D&D 新手快速創角工具

網站：<https://ginglemisa.github.io/dnd/>

作者：蔡秉衡 Reggie 瑞基

## 專案定位
這是一個 **非官方、粉絲製作** 的 D&D 新手創角工具。  
目標是讓第一次接觸 D&D 的玩家，能在手機上快速完成可用角色卡，盡快進到第一場遊戲。

本工具目前聚焦在 **2024 Basic Rules、等級 1～5** 的教學與入門使用。

## 使用情境
適合以下場景：
- TRPG 推廣活動
- 公眾場合現場教學
- 桌邊由 DM / 教學者帶新手創角
- 想先快速上手、再進一步學完整規則的玩家

## 功能簡介
- 手機友善的角色卡填寫流程
- 基本資料、能力值、技能、裝備、法術整合
- 自動計算技能與豁免
- 分享連結、QR、JSON 匯入 / 匯出
- localStorage 自動儲存
- 可切換「教學模式」與「完整角色卡模式」

## 為什麼只做到 1～5 級
這個工具的設計重點是「先幫新手開始第一場」。  
如果玩家能順利從 1 級玩到 5 級，通常已具備閱讀與使用更完整規則資源的基礎；
此時很推薦直接進入 Player’s Handbook 與完整角色工具。

## 使用方式
1. 開啟網站後，選擇「教學模式」或「完整角色卡模式」。
2. 新手建議使用教學模式，照步驟完成角色卡。
3. 完成後可用分享連結 / QR 提供給隊友，或匯出 JSON 備份。
4. 想離線使用時，可下載 `快速創角離線版.html`。
