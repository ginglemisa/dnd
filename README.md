# D&D 新手快速創角工具

https://ginglemisa.github.io/dnd/

## 專案定位
這是一個送給 D&D 新手玩家的小禮物。
目標很單純：讓大家更快完成角色建立，順利開始第一場冒險。

本工具以手機操作為主，並假設現場會有 DM 或教學者協助。

## 使用情境
本工具可作為商務帶團、付費教學或公開推廣活動中的輔助創角工具使用；但實務上仍請自行確認官方相關規範與風險。

## 功能簡介
- 針對新手流程設計的創角介面
- 支援角色資料本機儲存、匯入、匯出與分享連結
- 規則範圍聚焦在 **2024 Basic Rules 的 1～5 級內容**

### 為什麼只做到 1～5 級？
這個專案目標是「先把第一場玩起來」。
如果你已經從 1 級玩到 5 級，通常就會開始需要更完整的規則與角色選項。
