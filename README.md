# 版權說明

請不用擔心版權問題。

本網站使用的所有遊戲內容，都是 **海岸巫師（Wizards of the Coast）免費開放的基本規則內容**。

本工具只支援到 **5級（Level 5）**，原因很簡單：

如果玩家已經從 **1級玩到5級**，而且還想繼續玩 D&D，那就表示你已經準備好：

- 開始閱讀 **《玩家手冊》（Player's Handbook）**
- 使用更完整的 **角色卡工具或紙本角色卡**

這個網站的目標只是讓新手 **更容易開始第一場遊戲**。

# 使用方法
雖然老玩家應該不需要教學，但還是簡單寫一下使用順序。

**PS：**  
新手玩家不懂是很正常的，別緊張。  
作者預設 **新人在使用這個網頁時，旁邊會有主持人（DM）協助**。

如果你找不到主持人，可以到 **Facebook 搜尋「TRPG」相關社團**，通常都可以找到願意教學的玩家。

## 創角流程
1️⃣ **選擇基本資料**
- 職業
- 等級
- 背景
- 種族
- 陣營

2️⃣ **填入屬性**
- 手動填入屬性
- 或按下 **預設屬性按鈕**

3️⃣ **背景設定**
- 按背景調整屬性
- 下方選擇專長
- 技能頁打勾技能
- 選擇 **A 或 B 裝備**

4️⃣ **種族 / 法術**
- 種族或專長打勾技能
- 法術頁選擇 **施法屬性**
- 選擇法術

5️⃣ **基本數值**
填寫：
- 速度
- 先攻
- 被動感知

6️⃣ **計算數值**
按下：
- **[計算屬性豁免]**
- **[計算技能加值]**

7️⃣ **選擇裝備**
按照 **職業（A 或 B）或背景裝備選項**：
在裝備頁選擇：
- 主手武器
- 副手武器
- 護甲

8️⃣ **武器精通**
如果職業能力有 **精通（Mastery）**
在裝備頁下方：
- 點選精通屬性
- 選擇精通

9️⃣ **生成攻擊資料**
到 **動作頁**
按下： [獲取 主副手 武器資料]
完成。
